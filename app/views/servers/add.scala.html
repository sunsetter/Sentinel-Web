Add a new server

Adding a new server is surprisingly simple: You just add the sentinel agent to your server and run it.

No online configuration required, as soon as your server sends data, it will appear in the Sentinel Dashboard.

Now, when you have created your new account, you can proceed with next step: downloading and setting up Sentinel agent on your servers.
